Lesson 3: Display Property, Div, and Span

This lesson explained the difference between Block-level and Inline-level elements. I learned that block-level elements (like <p>, <h1>, or <div>) always start on a new line and take up the full width available. In contrast, inline-level elements (like <a>, <strong>, or <span>) flow within the line of text and don't force a new line.

This is important because it introduced the display property, which lets you change an element's default behavior (like making a block element display as inline). The lesson also defined the <div> tag as a generic block-level container and the <span> tag as a generic inline container, both of which are used to group content for styling.
