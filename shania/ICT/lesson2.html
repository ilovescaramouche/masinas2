Lesson 2: CSS Structure and Types

This lesson broke down the fundamental "how-to" of CSS. I learned the basic syntax of a CSS rule, which consists of a selector (the HTML element you want to style) and a declaration block. This block contains properties (like color) and values (like blue) separated by a colon.

The lesson's main focus was on the three ways to add CSS to a webpage:

  • Inline CSS: The style is placed directly inside an HTML tag using the style attribute. This is used for small, isolated changes.
  • Embedded (Internal) CSS: The styles are placed inside a <style> tag within the <head> section of the HTML file. This affects the entire page.
  • External (Linked) CSS: The styles are saved in a separate .css file. This file is then linked to any HTML page using a <link> tag in the <head>, making it the most efficient way to style an entire website.
