Lesson 1: Introduction to CSS

Overview

This lesson was a high-level introduction to Cascading Style Sheets (CSS). I learned that CSS is the language used to style and design webpages, controlling things like layout, colors, and fonts. It explained the "three layers" of a webpage:

  • Content Layer (HTML): Provides the basic structure.
  • Presentation Layer (CSS): Adds all the visual styling.
  • Behavior Layer (JavaScript): Handles user interaction and functionality.

The main benefit of CSS is that it separates content (HTML) from design (CSS), which makes websites much easier to maintain and update.
